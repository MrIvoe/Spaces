# Wiki Publish Guide

This repository stores wiki source files in `docs/wiki`.

## Publish to GitHub Wiki

1. Open repository wiki in browser: `https://github.com/MrIvoe/Spaces/wiki`
2. Create or edit pages with matching names:
   - Home
   - Build-and-Test
   - Plugin-Quickstart
   - Plugin-Development-Deep-Dive
   - Architecture
   - Settings-and-Persistence
   - Plugins-and-Themes
   - New-Solution-Playbook
   - Debugging-and-Troubleshooting
   - Release-Workflow
3. Copy markdown content from each file in `docs/wiki`.
4. Add sidebar content from `_Sidebar.md`.

## Suggested Cadence

- Update README first for consumer-facing changes.
- Update wiki pages for developer-facing implementation details.
- Keep release workflow page synced with actual release checklist.
