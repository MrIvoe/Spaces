# Premium Idle Theme Plugin

This plugin package is prepared for transfer to the standalone plugin repository.

## Intent
- Keep fences icon-first by default.
- Allow per-fence premium idle behavior policies:
  - roll up when not hovered
  - transparent when not hovered

## Notes
- Current app-side behavior toggles are now stored per fence in metadata.
- This plugin package is a starter scaffold so plugin-repo work can continue in isolation.
